# Presentation & Demo Materials

## Files

- `pitch_deck.pptx` — Presentation slides
- `video_demo.mp4` — Product demo video

## Pitch Deck Structure

1. **Title** — Project name + team
2. **Problem** — What problem, who has it?
3. **Solution** — Your solution
4. **Demo** — Screenshot / short video
5. **Architecture** — Simple system diagram
6. **Tech Stack** — Technologies used
7. **Traction** — Metrics, users, feedback
8. **Market** — Market size
9. **Team** — Who does what
10. **Ask** — What you need next

## Video Demo Checklist

- [ ] Introduce the problem
- [ ] Demo the main feature
- [ ] Show results
- [ ] Summarize impact
